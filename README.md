NAME
====

Rakuroonga - Raku bindings of Groonga

SYNOPSIS
========

```perl6
use rakuroonga;
```

DESCRIPTION
===========

Raku bidings for Groonga that provide full text search and column store features.
We can use Groonga's fast and highly functional features from Raku.
